# 合规与使用边界

这个项目先按个人本地工具设计，核心目标是个人备份、稍后阅读和非商业打印。

## 建议边界

- 只采集当前账号正常浏览时可见的公开内容。
- 不绕过登录、权限、付费墙、风控或访问限制。
- 不采集私密内容。
- 不采集评论区。
- 不做高频、高并发、规模化采集。
- 不把归档内容公开发布或商业分发。
- PDF 默认保留作者昵称、原始链接、平台来源和采集时间。

## 导出说明页建议

每份 PDF 可以自动加入说明页：

```text
本 PDF 由个人本地归档工具生成，仅用于个人备份、稍后阅读和非商业打印。
内容来源于小红书公开可见页面，版权归原作者及相关权利人所有。
请勿将本文件用于未经授权的公开传播、商业分发或二次出版。
```

## 后续如果商业化

如果未来从个人工具走向公开产品，需要重新评估：

- 平台服务条款。
- 内容版权授权。
- 用户账号和 Cookie 安全。
- 数据存储与隐私合规。
- 博主授权机制。
- 导出内容的使用限制和追责边界。
