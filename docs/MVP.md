# MVP 范围

## 必须支持

- 输入小红书博主主页链接。
- 打开本地浏览器，由用户手动完成登录。
- 自动滚动主页，收集当前可见的笔记列表。
- 逐条打开图文笔记，采集标题、正文、图片、发布时间和原始链接。
- 将博主信息、笔记数据和图片保存到本地归档目录。
- 支持断点续采和失败记录。
- 生成完整 PDF，默认按时间线排列。
- PDF 保留作者信息、来源链接和采集时间。

## 第一版暂不支持

- 评论区采集。
- 私密、付费或非当前账号可见内容。
- 高并发或多账号采集。
- 视频文件下载。
- AI 改写和深度总结。
- 在线打印下单。
- 多用户或云端同步。

## 可选增强

- 导出 Markdown。
- 按月份自动分章。
- 生成 HTML 预览页。
- 对视频笔记保存封面、标题、正文和原链接。
- 增量同步同一博主新增笔记。

## 成功标准

第一版完成后，应能对一个中等规模博主执行完整流程：

1. 采集至少几十篇公开可见内容。
2. 中断后可以继续。
3. 本地目录可读、可备份、可迁移。
4. PDF 能像一本书一样从头到尾阅读。
